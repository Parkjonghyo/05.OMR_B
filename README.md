# 05.OMR_B
2022 한울 팀프로젝트 OMR 사이트.
다른 요구사항 명세서는 root의 미니프로젝트.txt에 명시되어있다.

## 기능
기능추가 ▶ (로그인) 관리자 - 문제(10) 추가 삭제 수정 (시간이 되면..)

회원가입 ▶ 이름, id(중복확인), pw

대주제 ▶ 동물  
ㄴ소주제 ▶ 강아지, 고양이

통과 점수 ▶ 60점

시험 시간 ▶ 미정

로그인 창 시작 ▶ Login.jsp
